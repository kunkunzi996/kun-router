# Kun Cleanup Gate

`kun-cleanup-gate` 是从 Kun Coding Router V0.8.2 拆出的项目洁癖门，可独立调用。

## 安装

让 skills-manager 指向 `E:\kun-router\cleanup-gate`。本仓库只提供源码说明，不在此执行安装。

## 使用

直接说“执行洁癖门”、“帮我收尾”或“同步一下项目文档”。

## 边界

本 Skill 处理项目知识三层：项目文档与状态、项目 AI 规则、项目级 Agent 记忆。

它不处理已合并分支、worktree、临时数据等运行垃圾；这些归 Kun Coding Router 的流水线收口。
